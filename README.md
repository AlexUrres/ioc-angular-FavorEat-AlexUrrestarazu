Títol: FavorEat - Àlex Urrestarazu

Descripció: FavorEat és una aplicació per gestionar els nostres restaurants preferits, tant els ja visitats com els que ens han recomanat. L’aplicació permet organitzar-los fàcilment mitjançant filtres per preu, ubicació o tipus de cuina, facilitant així l'elecció del lloc ideal per a cada ocasió

Stack tècnic: Angular v18.2.12
              Node.js v20.20.1
              npm v10.8.2
              Standalone Components
              SCSS (Sass)
              Angular Routing
              Git  

Estat: Configuració inicial completada i verificada
